# 🚀 Playwright Automation Framework

This project is a UI automation testing framework built using Playwright and JavaScript.

## 📌 Purpose
- Practice automation testing skills
- Build real-world QA framework
- Learn Page Object Model (POM)
- Prepare for QA Automation Engineer roles

## 🛠 Tech Stack
- Playwright
- JavaScript
- Node.js

## 📁 Project Structure
- tests/ → Test cases
- pages/ → Page Object Models
- test-data/ → Test data files

## 🧪 Sample Test Scenarios
- Login automation
- UI validation
- Functional testing

## ▶️ How to Run
npm install  
npx playwright install  
npx playwright test
